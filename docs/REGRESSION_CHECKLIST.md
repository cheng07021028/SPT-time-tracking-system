# 回歸測試清單

每次修改交付前至少確認：

## 基本

- [ ] `python -m compileall .`
- [ ] `pytest -q`
- [ ] ZIP 無 `#Uxxxx` 亂碼檔名
- [ ] services 層沒有 import streamlit

## 01 工時紀錄

- [ ] 開始作業成功
- [ ] 完工作業成功
- [ ] 同 group_key 完工平均正確
- [ ] 不同 group_key 自動暫停舊作業
- [ ] 刪除後 01 不顯示
- [ ] 刪除後 02 不顯示 active
- [ ] Reboot 後資料不消失、不復活

## 03 製令管理

- [ ] 新增製令 Reboot 後存在
- [ ] 重複製令不可新增
- [ ] 有工時紀錄的製令不可刪除

## 04 人員名單

- [ ] 新增人員只新增一筆
- [ ] 工號唯一
- [ ] 停用後不可開始作業

## 06 / 11

- [ ] LOG 查詢不卡住
- [ ] 登入紀錄查詢不卡住
- [ ] 登入成功/失敗都有記錄

## 10 / 13

- [ ] 權限不足會阻擋
- [ ] 設定儲存後 Reboot 保留
- [ ] services 不使用 `st.session_state`
