# SPT Capacity System V2.3 修改檔案包

本包只包含本次有修改的檔案，不是完整專案包。

## 主要修正

1. 00 首頁標題改為「超慧科技製造部產能儀表板」。
2. 03 製造部組織圖改為樹枝圖：製造部 → 製一課 / 製二課 → 工段 → 人員。
3. 03 組織圖卡片縮小，人員卡只顯示姓名與職稱。
4. 03 組織圖支援拖拉課別、工段、人員卡片，並支援課別全螢幕放大/還原。
5. 01 超慧員工名單、02 派遣名單新增目前總人數科技卡片。
6. 04 產能負荷表新增「月別調整工時」權威資料表，調整工時會加總到需求總工時。
7. 04 產能表新增欄位：原始需求工時(h)、調整工時(h)、需求總工時(h)。
8. 09 情境模擬新增更直觀的人力缺口/餘裕科技卡片，並增加每月人力差異圖。
9. 修正「顯示圖表數據標籤」後圖表消失問題：改用安全字串標籤，不再使用容易造成 Plotly template 失敗的格式。
10. 每個表格的欄位設定按鈕統一為「套用設定」與「恢復設定」，並永久保存。
11. 修正 11 權限與系統設定中，快速新增帳號後表格仍顯示舊快取資料的問題。

## 覆蓋方式

將本包內檔案依相同路徑覆蓋到 GitHub 專案。
覆蓋後請到 Streamlit Cloud 執行：Manage app → Reboot app。

## 新增權威資料

本次新增資料表：

```text
data/persistent/authority/capacity_adjustments.json
```

系統會在第一次進入 04. 產能負荷表時自動建立 1月～12月的調整工時資料，不需要手動建立 JSON。
